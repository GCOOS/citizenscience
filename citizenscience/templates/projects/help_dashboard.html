Basics

When you click a bar from a bar chart, you will see only data in that year in a table.


Put mouse on a bar (not click), it will show the number of data entry.
To reset a filter (e.g. back to all years from 2012), click "reset"
If you want to filter 2012-2013, click 2013. This will add up 2013's data.

Time line
Mouse icon will be changed to "+" sign on Timeline filter. Click and drag to select the time range. You can slide the blue-colored area to change time range.

back to top

Data Table
Data Table will be dynamically changed by filters.

With this option, you can change row numbers. Default number is 10. When you change, you can see more entries without changing a page at the bottom of the table.

Pages are creted based on the number of entries in a table. When you change the number of entries, the pagination automatically run and divided tables.

Arrange rows in ascending/decending order.

With these buttons, you can copy all entries to clipboard, save as CSV file, and print them out.

Please use browser's print function to print out. If you want to add more fields, turn on a specific field you desire to show/print out.

The default settings hide many fields which will not be shown when you print the table out. However, when you copy to clipboard or save in a CSV file, all filelds will be exported regardless of show/hide condition.

back to top

Map
Initially it shows the number of all entries. After filtering out, the number will change dynamically as well as map. To reset ALL filters, click the button.
Table will not be changed when you zoom in/out a map: Only filters (bar chart and time line) will change data table and points in a map.

To go back to default map extent, click a house icon.

There are three base layers. Default setting is Light theme.
